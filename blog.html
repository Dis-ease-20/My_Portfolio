Why HTML 5??

What is CSS 3?

Jan 2021

Created by Hakon Wium Lie in Oct 1994, CSS stands for Cascade Style Sheet. CSS is one of the most important elements and languages used to create websites. Just as much as HTML, CSS plays the role of decorating and styling the website to make it look more professional and appealing to the user. In this field you will see job titles such as UI designers which purely focuses on the look and feel of the website. CSS uses its own syntax such as curly braces and semi-colons rather than tags like HTML. CSS however does require more practice and understanding because it has a lot of properties to change things like resizing images, increasing text size and color, to positioning things in the right place and not forgetting having a good eye for design, color scheme and layouts.



Why CSS 3?


The evolution of CSS is much like HTML, remember that these two technologies will always go hand in in hand in web development. As HMTL improves so does CSS, the third version of CSS include very interesting properties that have animations, rotations, pop ups without using JavaScript. On of the popular proprieties in CSS is Flex Box and CSS Grids.

What is Web Development?

Jan 2021

Web development is basically the application, discipline and creation of building modern websites. There a re a few fundamental principles when going into web development, here are they:

· Client side (Front end):

This means everything you see on the web browser, this deals with the look and feel, the functionality and special features the website has.

· Server side (Back end):

This means everything that happens in the background/ behind the scenes, all the functionally that you may not see such as requesting a certain website and connecting to databases.

· Database:

A database is a tool that is used to store user information such as login details, authentication and security.

Now that we understand the principles, there are 3 types of web developers, which are:

1. Frontend web developer
2. Backend web developer
3. Full-stack web developer

Explaining the differences shall be discussed in a much later article. One of the key attributes to become a web developer is to have a strong eye of design principles such as layout and color schemes as well as has having strong understanding of code logic. The main languages for web developers for beginners would be HTML 5, CSS 3 and JavaScript, as you progress you can pick up frameworks and libraries like React, Angular, PHP, Python and many more.




Do you need a degree to become a web developer?


Having a degree/diploma is recommended and will place you at a very good advantage when applying for jobs, this just gives companies more confidence that you understand some programming principles and concepts such as Object Orientated Programming (OOP) but with the rapid increate in technology and more companies going online, becoming a web developer is one of the easiest ways to get into the tech space and you do not require a degree or diploma, one of the biggest perks of becoming a web developer in 2021 is the international opportunities, remote work and really good paying salaries.
